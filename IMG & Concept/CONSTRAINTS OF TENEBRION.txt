TENEBRION_CUST_FREQ=0/1
- If Enabled then ignore all Min,Mid,Max CPUFreq. It will use:

TENEBRION_CUST_FREQ_[0-Max]_MIN=[Value]
TENEBRION_CUST_FREQ_[0-Max]_MID=[Value]
TENEBRION_CUST_FREQ_[0-Max]_MAX=[Value]

This will poll using /sys/devices/system/cpu/cpufreq/policy*

TECNO-LH8n:/sys/devices/system/cpu/cpufreq # ls
policy0  policy6  schedhorizon

Example:

On policy0
TENEBRION_CUST_FREQ_0_MIN=968000
TENEBRION_CUST_FREQ_0_MID=1393000
TENEBRION_CUST_FREQ_0_MAX=1916000

On policy6
TENEBRION_CUST_FREQ_6_MIN=898000
TENEBRION_CUST_FREQ_6_MID=1418000
TENEBRION_CUST_FREQ_6_MAX=2306000

The list of detecting screen state, this need to be done during installation (customize.sh) so we don't keep polling each time execution.

Determine where the path we will use:

Common Path
"/sys/class/drm/card0-DSI-1/dpms" - "On" or "Off"

Second Path
"/sys/class/leds/lcd-backlight/brightness" - Off then value is 0, if on the value more than 0

Compability using cmd
"cmd deviceidle get screen" - "false" or "true"

Therefore the constraints defined is 

TENEBRION_COMMON_PATH=0/1 
(For "/sys/class/drm/card0-DSI-1/dpms" - "On" or "Off")

TENEBRION_SECOND_PATH=0/1
("/sys/class/leds/lcd-backlight/brightness" - Off then value is 0, if on the value more than 0)

TENEBRION_COMPABILITY=0/1
("cmd deviceidle get screen" - "false" or "true")
